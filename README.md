# ConvNets and Seismic Denoising

A collection of computer vision experiments covering convolutional neural networks implemented from scratch and a practical seismic image denoising task.

## Contents

- [`models_from_scratch`](models_from_scratch) — implementations and training experiments with classic convolutional architectures, including AlexNet, VGG, and ResNet.
- [`seismic_denoising`](seismic_denoising) — experiments with convolutional models for removing noise from seismic images.

The repository combines architecture-level experiments with an applied computer vision problem from the seismic domain.
