# ConvNets and Seismic Denoising

A collection of computer vision experiments with convolutional neural networks, image segmentation, and seismic image denoising.

## Projects

- [`models_from_scratch`](models_from_scratch) — CNN implementations, training utilities, and architecture experiments with AlexNet, VGG, ResNet, EfficientNet, and U-Net.
- [`segmentation_notebooks`](segmentation_notebooks) — semantic segmentation experiments with U-Net, residual U-Net, and DeepLabV3.
- [`seismic_denoising`](seismic_denoising) — U-Net experiments for removing noise from seismic images.
