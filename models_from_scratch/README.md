# Convolutional Models from Scratch

Implementations and training experiments with convolutional neural network architectures.

The project covers AlexNet, VGG, ResNet, EfficientNet, and U-Net. The classification experiments use a 10-class subset of ImageNet and focus on model components, training behavior, and architectural differences.

## Contents

- `train_cnn_1.ipynb` — training and evaluation of AlexNet, VGG, and ResNet.
- `models/` — model implementations.
- `notebooks_with_experiments/` — additional architecture experiments.
- `results/` — experiment and evaluation notebooks.
- `utils/` — dataset, training, inference, and visualization utilities.
